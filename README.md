## Overview

This repository is a portfolio of computer vision projects covering the full spectrum of visual AI -- from classical image processing to deep learning-powered detection and generation. Each project includes theory, implementation, and evaluation with real-world datasets.

The goal is to develop expertise in how machines perceive, process, and generate visual information using state-of-the-art architectures and techniques.

## Getting Started

### Prerequisites

```bash
pip install torch torchvision opencv-python albumentations matplotlib jupyter
# For TensorFlow projects:
pip install tensorflow
```

### Running Notebooks

```bash
jupyter notebook
```

Navigate to the project folder of interest and open the corresponding `.ipynb` file. GPU is recommended for training-heavy notebooks.
